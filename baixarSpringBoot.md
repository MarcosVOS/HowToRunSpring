# Como baixar um projeto Spring

### 1. Acesse o site 
### https://start.spring.io/
![alt text](https://github.com/MarcosVOS/HowToRunSpring/blob/main/docs/img1.png)

### 2. Selecione a opção Maven
![alt text](https://github.com/MarcosVOS/HowToRunSpring/blob/main/docs/img2.png)

### 3. Defina o nome do projeto (no exemplo, mantemos o valor padrão)
![alt text](https://github.com/MarcosVOS/HowToRunSpring/blob/main/docs/img3.png)

### 4. Selecione a versão Java 21 
![alt text](https://github.com/MarcosVOS/HowToRunSpring/blob/main/docs/img4.png)

### 5. Clique no botão Add Dependencies
![alt text](https://github.com/MarcosVOS/HowToRunSpring/blob/main/docs/img5.png)

### 6.Um modal será aberto, como este: 
![alt text](https://github.com/MarcosVOS/HowToRunSpring/blob/main/docs/img6.png)

### 7. No campo de busca, digite spring web 
![alt text](https://github.com/MarcosVOS/HowToRunSpring/blob/main/docs/img7.png)

### 8. Selecione a opção Spring Web 
![alt text](https://github.com/MarcosVOS/HowToRunSpring/blob/main/docs/img8.png)

### 9. Após selecionar, a dependência aparecerá na lista assim: 
![alt text](https://github.com/MarcosVOS/HowToRunSpring/blob/main/docs/img9.png)

### 10. Clique no botão Generate 
![alt text](https://github.com/MarcosVOS/HowToRunSpring/blob/main/docs/img10.png)

### 11. Será solicitado que você escolha onde deseja salvar o arquivo 
![alt text](https://github.com/MarcosVOS/HowToRunSpring/blob/main/docs/img11.png)

### 12. Após escolher o local, será baixado um arquivo .zip com o nome definido na etapa 3
![alt text](https://github.com/MarcosVOS/HowToRunSpring/blob/main/docs/img12.png)
