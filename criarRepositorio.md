# Requisito Criar uma conta no github 
[Criar conta no github](https://github.com/signup?source=login)

### 1. Após a criação da conta acessar 
[Acessar a tela inicial do github](https://github.com/)

### 2. Na tela home clicar no botão new  
![alt text](https://github.com/MarcosVOS/javaSpringBootTemplate/blob/main/docs/img13.png)

### 3. Após isso ira aparecer uma tela assim   
![alt text](https://github.com/MarcosVOS/javaSpringBootTemplate/blob/main/docs/img14.png)

### 4. Após isso em Repository name definir o nome do do repositorio   
![alt text](https://github.com/MarcosVOS/javaSpringBootTemplate/blob/main/docs/img15.png)

### 5. Após escrever o nome do repositorio (no exemplo coloquei demo) clicar no botão Create Repository   
![alt text](https://github.com/MarcosVOS/javaSpringBootTemplate/blob/main/docs/img16.png)

### 6. Após criar o repositorio ira aparecer uma tela assim   
![alt text](https://github.com/MarcosVOS/javaSpringBootTemplate/blob/main/docs/img17.png)

### 7. Agora clicar no botão no link "uploading an existing file."   
![alt text](https://github.com/MarcosVOS/javaSpringBootTemplate/blob/main/docs/img18.png)

### 8. Ira aparecer uma tela assim    
![alt text](https://github.com/MarcosVOS/javaSpringBootTemplate/blob/main/docs/img19.png)

### 9. Após isso pegar o arquivo que foi baixado     
![alt text](https://github.com/MarcosVOS/javaSpringBootTemplate/blob/main/docs/img12.png)

### 10. Pegar o arquivo que foi baixado e anexar nessa parte     
![alt text](https://github.com/MarcosVOS/javaSpringBootTemplate/blob/main/docs/img19_1.png)

### 11. Após anexar o arquivo deve aparecer assim     
![alt text](https://github.com/MarcosVOS/javaSpringBootTemplate/blob/main/docs/img20_1.png)

### 12. Após anexar o arquivo cicar no botão Commit Changes     
![alt text](https://github.com/MarcosVOS/javaSpringBootTemplate/blob/main/docs/img21.png)

### 13. Caso tudo tenha dado certo ira aparecer uma tela assim     
![alt text](https://github.com/MarcosVOS/javaSpringBootTemplate/blob/main/docs/img22_1.png)
